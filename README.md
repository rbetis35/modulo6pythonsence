# 🤖 Proyecto Integrador: Machine Learning (Módulo 6)
### Diplomado en Fundamentos de Análisis de Datos — SENCE

## 📝 Descripción
Este proyecto corresponde al **Módulo 6** y tiene como objetivo aplicar un flujo básico de **Machine Learning supervisado** utilizando **Regresión Lineal** para predecir el gasto total de clientes en un escenario de E-commerce.

El trabajo se desarrolló en un **Jupyter Notebook** e incluye:
- carga y selección de variables,
- división de datos en entrenamiento y prueba,
- entrenamiento de un modelo,
- evaluación con métricas,
- visualización de resultados,
- y una predicción de ejemplo para un cliente nuevo.

## 🎯 Objetivo del Modelo
Predecir el valor de `Monto_Total` utilizando las variables:
- `Edad`
- `Total_Compras`
